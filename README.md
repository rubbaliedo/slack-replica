# slack-replica

A flexible slack-replica implementation.

## Features
- Real-time messaging
- User authentication
- Channel creation

## Technology Stack
- Node.js
- Socket.io
- Express

## Usage
1. Clone the repository: `git clone https://github.com/rubbaliedo/slack-replica.git`
2. Install dependencies: `npm install`
3. Start the server: `npm start`
